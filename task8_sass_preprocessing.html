Sass
Preprocessed.

Variables, mixins, nesting, and compilation — the Sass workflow that elevates plain CSS into a maintainable, scalable design system.

$

Sass Variables

Define colors, sizes, and font-stacks once in $vars — reuse everywhere. Change a value in one place and the entire design updates.

@

Mixins

Reusable blocks of CSS logic. @mixin + @include eliminates copy-paste and makes patterns composable.

{ }

Nesting

Write child selectors inside their parent, mirroring your HTML hierarchy for better readability and less repetition.

⚙

Compilation

Sass compiles styles.scss → styles.css. The browser only ever sees clean, standard CSS.

01

Variables

Store values like $brand-color once and reference them throughout the stylesheet.

02

Nesting

Mirror DOM structure in your CSS. The & symbol references the parent selector.

03

Mixins

Parameterisable blocks of CSS. Pass arguments to generate different outputs from a single definition.

04

Partials

Split your CSS into _variables.scss, _mixins.scss files and @use them.

Source vs Compiled

The left panel shows the Sass source. The right panel shows the CSS output after compilation.

styles.scss — Sass Source 
// ── Variables
$accent-pink: #f72585;
$accent-cyan: #4cc9f0;
$bg-surface:  #13131c;
$border:      #1e1e2e;
$radius:      10px;
$transition:  all 0.25s ease;

// ── Mixin definition
@mixin card-base {
  background: $bg-surface;
  border: 1px solid $border;
  border-radius: $radius;
  padding: 32px;
}

// ── Nested selector with &
.feature-card {
  @include card-base;
  transition: $transition;

  // & = .feature-card
  &:hover {
    border-color: $accent-pink;
    transform: translateY(-4px);
  }

  .card-icon {
    width: 48px;
    height: 48px;
  }

  h3 { color: $accent-cyan; }
  p  { color: #6a6880; }
}

// ── Compile: sass styles.scss styles.css
styles.css — Compiled Output 
/* compiled from styles.scss */

.feature-card {
  background:   #13131c;
  border:       1px solid #1e1e2e;
  border-radius:10px;
  padding:      32px;
  transition:   all 0.25s ease;
}
/* $vars resolved, @include expanded */

.feature-card:hover {
  border-color: #f72585;
  transform:    translateY(-4px);
}
/* &:hover → .feature-card:hover */

.feature-card .card-icon {
  width:  48px;
  height: 48px;
}
/* nested .card-icon expanded */

.feature-card h3 {
  color: #4cc9f0;
}
.feature-card p {
  color: #6a6880;
}
/* all $vars replaced with values */
