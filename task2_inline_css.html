CSS

Level 1 · Beginner

Style Is Power.

Inline CSS places style attributes directly on HTML elements — no separate stylesheet needed. It's raw, direct, and perfect for understanding how CSS fundamentally works.

Property #01

Font Color

The color property sets the text color of an element.

style="color: #ff4d6d"

style="color: #4dffd2"

style="color: #e5fc3c"

Property #02

Font Size

The font-size property scales the text to any measurement.

font-size: 12px — Small

font-size: 18px — Medium

font-size: 28px — Large

Property #03

Background Color

The background-color property fills an element's background.

background-color: #e5fc3c
background-color: #4dffd2
background-color: #ff4d6d
index.html · Inline CSS Applied 
<!-- Inline CSS applied directly to HTML elements -->

<h1 style="color: #0e0e14; font-size: 48px; background-color: #e5fc3c;">
  Hello, Cognifyz!
</h1>

<p style="color: #4dffd2; font-size: 18px;">
  This paragraph has teal-colored text at 18px.
</p>

<div style="background-color: #ff4d6d; color: #fff; padding: 16px;">
  A div with a red background and white text.
</div>
